BattleShip Game

The Battleship Game is a Java console-based two-player strategy game. Each player places their ships on a grid and then takes turns firing at their opponent's grid to sink their ships. The first player to sink all the opponent's ships wins.

How to Play
Players take turns entering coordinates to fire shots at their opponent's grid.
The game provides feedback on hits and misses, and displays the current state of both players' grids. The game ends when one player sinks all the opponent's ships.

Features
Two-player interactive gameplay. Players can place their ships on the grid. Players take turns firing shots at their opponent's grid.
Visual display of the game board with ships and shots. Detects when a player has sunk all opponent's ships and declares the winner.

How to Place Ships
Players take turns placing their ships on their grid. Players input the coordinates for the start and end of the ship (e.g., A1 A5 or B5 E5).
The game validates the coordinates and adds the ship to the grid if valid. Different ship types have different lengths: Aircraft Carrier (5 cells), Battleship (4 cells), Cruiser (3 cells), Submarine (3 cells), and Destroyer (2 cells).

Technologies Used
Java for the core game logic. Command-line interface for input and display.

Coffee Machine

This Coffee Machine Simulator is a command-line program that allows users to interact with a virtual coffee machine. Users can buy different types of coffee, fill the machine's resources, take money from the machine, and check the remaining resources.

Features
- Buy espresso, latte, or cappuccino with customizable sugar cubes.
- Refill the machine's water, milk, coffee beans, disposable cups, and sugar.
- Take the money from the machine's earnings.
- View the current status of the machine's resources.
- Interactive user experience through the command-line interface.

How to Run
  • Clone or download this repository to your local machine.
  • Open your terminal or command prompt.
  • Navigate to the directory containing the Coffee Machine Simulator files.
  • Run the program by executing the appropriate command


Usage
  • Start the program and follow the on-screen instructions.
  • Choose from options like buying coffee, refilling resources, taking money, checking resources, and exiting the program.
  • When buying coffee, you can choose the type and optionally add sugar cubes.
  • When refilling, provide the quantities of water, milk, coffee beans, disposable cups, and sugar.
  • To take money, you'll need to enter a password.
  • The "remaining" option displays the current status of the coffee machine's resources.


Technologies Used
- JavaScript for logic and interactivity.
- `readline-sync` for handling user input in the command line
